# Me, Inc. 個人營運帳本

> 把你的人生，當作一間公司在經營

## 🏢 設計理念

| 公司術語 | 個人意義 |
|---|---|
| 營運收入 Revenue | 你的所有收入（薪資、副業、投資⋯） |
| 各部門支出 | 食 🍱、衣 👔、住 🏠、行 🚗、娛 🎬、其他 📂 |
| 本月淨利 Net Profit | 收入 − 支出 = 你這個月「賺」了多少 |
| 淨利率 Margin | 儲蓄率（你的「公司」是否健康） |
| 燃燒率 Burn Rate | 每月平均花費 |
| 跑道 Runway | 沒收入還能撐幾個月 |

## 🚀 完整安裝流程（首次約 5 分鐘）

### Phase 1：建立後台（Google Sheet + Apps Script）

> 💡 建議**新建一份 Google Sheet** 給個人帳本用，與顧問版資料分開。

1. 打開 **https://sheets.new** 建立新試算表，命名「Me, Inc.」
2. 上方選單：**擴充功能 → Apps Script**
3. 全選刪除預設內容 → 貼上 `google-apps-script.gs` 完整內容 → 儲存
4. 右上「**部署 → 新增部署作業**」→ 類型：**網頁應用程式**
5. 設定：
   - 執行身分：**我（您的 email）**
   - 存取權：**所有人 Anyone** ⚠️（不是「擁有 Google 帳戶的所有人」）
6. 部署 → 授權 → 複製「**網頁應用程式網址**」

### Phase 2：本機測試（電腦先確認可用）

7. 雙擊 `me-inc-ledger.html` 在瀏覽器開啟
8. 在設定畫面貼上剛才的網址 → 「開始營運」
9. 進入主畫面，試著新增一筆支出測試

## 📱 變身 iPhone App（PWA 加入桌面圖示）

由於 iOS Safari 的限制，**file:// 開啟的網頁無法「加入桌面」**。你必須先把 HTML 放到網路上（HTTPS）才能變 PWA App。

最快、免費、無需技術背景的方式：**Netlify Drop**

### 操作步驟

1. **電腦上**打開 **https://app.netlify.com/drop**
2. **拖拉** `me-inc-ledger.html` 到頁面上的虛線框
3. 等 10 秒，網站建立完成，你會得到一個網址，例如：
   ```
   https://gleaming-cupcake-abc123.netlify.app
   ```
4. **手機上**打開 Safari，輸入這個網址
5. 點底部「**分享**」按鈕（中間那個 ⬆ 圖示）
6. 滾動找到「**加入主畫面**」
7. 點「**新增**」

完成！桌面上會出現「Me, Inc.」圖示，點開像原生 App 一樣全螢幕運行。

> 🔒 **想要更隱私？** Netlify 預設網址是公開的（雖然網址很難猜）。如果想保護：
> 1. 註冊免費 Netlify 帳號
> 2. 在站點設定 → Visitor access → 設密碼保護
> 3. 或改用 Cloudflare Pages、Vercel 等其他免費託管

## ⚡ 功能特色

### 主畫面（儀表板）
- **本月淨利 Hero Card**：最醒目的數字，含淨利率徽章
- **營運收入 / 營運支出**：本月對照
- **部門支出佔比**：六大部門進度條，點擊可快速篩選
- **月燃燒率**：近 3 個月平均支出
- **公司跑道**：以目前現金除以燃燒率算出，<3 個月會變紅警示

### 記帳體驗
- 右下角懸浮「+」大按鈕，單手可按
- 表單從下方滑入（Bottom Sheet）
- 數字輸入時 iPhone 自動彈出數字鍵盤
- 大型類別按鈕，含 emoji，一眼可選
- 點交易明細可編輯/刪除

### 離線支援
- 沒網路也能新增/刪除/修改記錄
- 操作會暫存到佇列
- 連線恢復時自動同步到 Google Sheet
- 上方有 ☁ / ☁ ✕ 圖示顯示連線狀態

### 月份切換
- 頂部箭頭切換月份，回顧過去
- 永遠以「本月」為預設

## 🔧 常見問題

### Q1：可以沿用顧問版的 Apps Script 嗎？
**可以，但建議分開**。Apps Script 程式碼相同（通用 CRUD），但類別不同會混在一起。建議新建 Sheet + 新部署。

### Q2：個人版的類別可以加減嗎？
可以！打開 HTML 找到 `INCOME_CATS` 和 `DEPARTMENTS` 常數，依格式增減即可。每個項目有 `id`、`name`、`emoji`、`color` 等屬性。

### Q3：iPhone 加入桌面後沒網路會怎樣？
- ✅ App 可以正常開啟（已快取）
- ✅ 可以新增記錄（暫存到佇列）
- ✅ 可以查看本機快取的舊資料
- 連線恢復時自動同步

### Q4：可以匯出資料嗎？
直接打開 Google Sheet → 檔案 → 下載 → Excel。所有資料都在 Sheet 裡，永遠掌握在自己手上。

### Q5：和家人共用？
把 Google Sheet 分享編輯權給家人，他們也用同一個網址即可。不過目前沒有「誰記的」標記。

## 🛡 隱私與安全

- 所有資料都儲存在**你自己的 Google Sheet**，不經過任何第三方伺服器
- API 網址相當於密鑰，請勿公開分享
- Netlify 免費版網址技術上是公開的，但網址很難猜中
- 如需更強隱私，可在 Netlify 加密碼保護

## 💡 進階使用建議

1. **每天記帳 1 分鐘**：晚上睡前打開 App，把當天花的錢輸入完
2. **每週末檢視儀表板**：看看哪個部門花得最多，是否符合自己期待
3. **每月初設目標**：例如「下個月娛樂部支出降到 X 元以下」
4. **盯著「淨利率」**：>20% 健康、10-20% 普通、<10% 該注意、<0% 警報
5. **看「跑道」做決策**：跑道 > 6 個月可以考慮做投資、跑道 < 3 個月該緊縮支出
